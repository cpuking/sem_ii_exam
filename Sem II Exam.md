## 🧠 Universal Answer Writing Formula

For **every answer** in the exam, use this structure:

```
1️⃣ DEFINITION (1–2 sentences)
   "X is defined as..."

2️⃣ KEY POINTS (bullet points, 3–5 points)
   - Point 1
   - Point 2
   - Point 3

3️⃣ TABLE/DIAGRAM (if applicable)
   | Column A | Column B |

4️⃣ CODE EXAMPLE (for technical questions)
   ```language
   code here
   ```

5️⃣ ADVANTAGES / SUMMARY (1–2 lines)
```

> ⚡ A code snippet = automatic +1 to +2 marks  
> ⚡ A table = marks for "comparison" questions  
> ⚡ A diagram = bonus marks for architecture questions
