# 🧠 6. Answer Writing Trick (VERY IMPORTANT)

Use this structure:

```
1. Definition
2. Explanation
3. Diagram (if possible)
4. Example
5. Conclusion
```

---

### Example:

Instead of:  
“Node.js is runtime”

Write:

- Definition
    
- Features
    
- Event loop diagram
    
- Example
    

👉 This increases marks by 2–3 instantly
