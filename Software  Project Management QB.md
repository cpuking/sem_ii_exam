# ✅ 🔹 UNIT 1: BASICS OF PROJECT MANAGEMENT

---

## ⭐ 1. What is a Project?

**Answer:**

A project is a **temporary endeavor** undertaken to create a **unique product, service, or result**.  
It has:

- Defined start and end
    
- Specific objectives
    
- Unique outcomes
    

**Examples:**

- Developing a mobile app
    
- Building a website
    

👉 As stated in your notes, projects are different from routine work because they are **temporary and goal-oriented**

---

## ⭐ 2. What is Project Management?

**Answer:**

Project management is the application of:

- Knowledge
    
- Skills
    
- Tools
    
- Techniques
    

to complete a project within:

- Scope
    
- Time
    
- Cost
    

It involves:

- Planning
    
- Execution
    
- Monitoring
    
- Control
    

---

## ⭐ 3. Systems View of Project Management

**Answer:**

A systems view considers the project as a **complete system** where all components interact.

### Components:

- Inputs → resources
    
- Processes → planning, execution
    
- Outputs → deliverables
    

👉 Helps manage complexity and risks

---

## ⭐ 4. Organizational Structures

**Answer:**

### Types:

1. **Functional**
    
    - Department-based
        
    - Low PM authority
        
2. **Matrix**
    
    - Dual reporting
        
    - Balanced control
        
3. **Projectized**
    
    - PM has full control
        
4. **Composite**
    
    - Mix of structures
