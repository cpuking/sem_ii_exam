## 🧠 Answer Writing Formula

For every answer, use this structure:

```
1. Definition (1–2 lines)
2. Key Points / Explanation (bullet points)
3. Diagram / Table (if applicable)
4. Code Example (for technical topics)
5. Advantages / Disadvantages
```

> ⚡ Even a small code snippet = +1 to +2 marks
